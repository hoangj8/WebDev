Just a quick set of instructions on how to setup/use Git Bash in a Windows environment.

Setting up a new repository

  1. Install Git Bash
  2. Login to Github.com and create a new remote repository
  3. In the new repo's setup page copy the link
  4. Open up Git Bash and navigate to the whichever directory you want your project to be in. You can get to different drives by using /c or /d
  5. Initialize the current directory as a local git repository with git init -b main
  6. Add the files that you'd like to keep track of to the staging area with git add <filename.
  7. Commit the changes to local repo with git commit -a -m "summary".
  8. Connect the local repo to the remote repo with git remote add origin remote repo url.
  9. Push the changes to the remote repo with git push

Workflow

  1. Pull any changes from remote repo
  2. Use git status to keep track of any changes you've made so far
  3. Use git add filename to add any new files created since the last push to remote repo
  4. Use git commit -a -m "summary" to send changes to local repository
  5. Use git push to add changes to remote repo.
